<h1 align="center">🍕 Yummy - Online Food Delivery Web Application</h1>

<p align="center">
Modern Java Full Stack Food Delivery Application
</p>

Yummy is a modern online food delivery web application developed using Java Full Stack technologies.
The application allows users to browse restaurants, search food items, manage their shopping cart, place orders, and view previous orders through an attractive responsive interface.
This project was built as a Java Full Stack learning project using JSP, Servlets, JDBC, MySQL, HTML, CSS and JavaScript.

## ✨ Features
- 👤 User Registration & Login
- 🍽 Browse Restaurants
- 🍕 Browse Food Menu
- 🔍 Search Food
- 🛒 Add to Cart
- ➕ Increase / Decrease Quantity from cart
- ❌ Remove Items from Cart
- 💳 Checkout
- 📦 Place Orders
- 📋 My Orders
- 👤 User Profile
- 📱 Responsive Design
- 🔒 Session Management

## 🏗 Architecture
This project follows the MVC (Model-View-Controller) architecture.
- Model → Java Beans
- View → JSP, HTML, CSS, JavaScript
- Controller → Java Servlets
- Database → MySQL

The application is fully dynamic, with all restaurant, menu, user, cart, and order data retrieved from and stored in a MySQL database using JDBC.

## 🚀 How to Run

1. Clone the repository
```
git clone https://github.com/sai9380/Yummy.git
```
2. Import the project into Eclipse
3. Configure Apache Tomcat
4. Import the MySQL database
5. Update database credentials in
```
DbConnection.java
```
6. Run the project
7. Open
```
http://localhost:8080/Yummy
```
