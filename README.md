<h1 align="center">ЁЯНХ Yummy - Online Food Delivery Web Application</h1>

<p align="center">
Modern Java Full Stack Food Delivery Application
</p>

Yummy is a modern online food delivery web application developed using Java Full Stack technologies.
The application allows users to browse restaurants, search food items, manage their shopping cart, place orders, and view previous orders through an attractive responsive interface.
This project was built as a Java Full Stack learning project using JSP, Servlets, JDBC, MySQL, HTML, CSS and JavaScript.

## ЁЯЪА How to Run

1. Clone the repository
```

```
2. Import the project into Eclipse
3. Configure Apache Tomcat
4. Import the MySQL database
5. Update database credentials in
```

```
